# 快速开始指南

这份仓库最值得先跑通的是 [`app.py`](../../app.py)，不是早期的加密回测脚本。

## 你应该怎么进入这个项目

### 如果你想看“当前主功能”

跑财务规划器：

```bash
pip install -e .
pip install streamlit plotly
streamlit run app.py
```

说明：

- `pip install -e .` 只安装量化工具链依赖
- `streamlit`、`plotly` 没写进 `pyproject.toml`
- `scikit-image` 是可选依赖，缺失时只会影响部分敏感性分析细节

打开后你会看到：

- 参数侧边栏
- 破产风险、债务清零时间、净资产分位数
- 现金、债务、投资、职业状态图
- 第 10 年总资产敏感性分析
- 汇总 CSV 下载

### 如果你想看“历史量化研究线”

推荐顺序：

1. 先看 [`notebooks/`](../../notebooks)
2. 再看 [`src/data_handler/`](../../src/data_handler) 和 [`src/backtest/`](../../src/backtest)
3. 最后再看脚本和测试

原因很简单：这个仓库后期很多真实工作都已经搬到 notebook 里了。

## 不建议你一开始就做的事

- 不建议直接相信旧版 README 或旧教程中的 API 名称
- 不建议直接把 `script/run_backtest.py` 当成开箱即用入口
- 不建议先看 `docs/notes/`，那更像个人知识库，不是系统文档

## 一句话判断

- 想理解现在的项目：从 `app.py` 开始
- 想理解早期量化研究：从 `notebooks/` 开始
- 想清理仓库：看 [`05-进阶功能.md`](./05-进阶功能.md)
